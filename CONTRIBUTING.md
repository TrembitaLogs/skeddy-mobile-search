# Contributing

This repository is published for **transparency and reference only**.

## External contributions are not accepted

Pull requests opened from forks are automatically closed without review. Please do not invest time preparing PRs against this repository — they will not be merged.

## Reporting a security vulnerability

Issue tracking is disabled on this repository. If you have discovered a security vulnerability and wish to report it responsibly, please email **trembitalogs@gmail.com** with details. Do not disclose publicly until the issue is acknowledged.
