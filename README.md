# Skeddy Search

Device-side Android app for the Skeddy driver platform — runs on a dedicated device, polls the Skeddy backend for ride offers, applies driver-defined filters, and surfaces matching offers in real time.

## Stack

Kotlin · Android (minSdk 28, targetSdk 34) · Coroutines · Room · WorkManager · Retrofit · OkHttp · kotlinx.serialization · EncryptedSharedPreferences · Play Services Location · Material · GitHub Actions · DigitalOcean Spaces

## Status

Source published for transparency. Closed to external contributions.
